## 📖 产品简介

**	Alvin 数页**是一款专为小学生设计的智能数学练习题生成器。您可以根据孩子的学习进度，灵活设置难度等级、运算类型、计算范围和出题形式，一键生成个性化数学练习试卷，支持保存为 PDF 或直接打印。

## ✨ 主要功能

- 🎯 **智能难度控制** — 三级难度（容易/中等/困难），精准匹配学习水平
- ➕ **加法** — 支持进位控制，可设定是否有进位
- ➖ **减法** — 支持借位控制，结果保证为正整数
- ✖️ **乘法** — 支持表内乘法（九九乘法表）和普通乘法
- ➗ **除法** — 支持表内除法、表内余数、一般余数三种模式
- 🔀 **混合运算** — 组合多种运算符，遵循先乘除后加减规则
- 📏 **横式计算** — 每页 20 题，2 列 × 10 行排列
- 📐 **竖式计算** — 每页 12 题，3 列 × 4 行排列
- 🔄 **自动去重** — 同一次生成绝不出现重复题目
- 💪 **难度最大化** — 智能筛选同范围内难度最高的题目
- 📄 **PDF 导出** — 一键保存为 PDF 文件，支持打印
- 🔢 **多页生成** — 可自定义生成 1～N 页

## 🚀 使用步骤

### 第一步：选择难度等级

在首页的 **"难度等级"** 卡片中，选择适合的难度：

| 难度     | 说明                                | 适用对象       |
| -------- | ----------------------------------- | -------------- |
| **容易** | 强制无进位/无借位，适合入门         | 一二年级初学者 |
| **中等** | 约 65% 概率出现进位/借位            | 二三年级学生   |
| **困难** | 强制 100% 进位/借位，题目最具挑战性 | 三四年级提高   |

### 第二步：选择运算方法

在 **"运算方法"** 卡片中，选择一种或多种运算：

**基础运算模式**（可多选）：

- **加法** — 单一或组合使用
- **减法** — 单一或组合使用
- **乘法** — 可额外选择"表内乘法"计算类型
- **除法** — 可额外选择"表内除法"、"表内余数"、"一般余数"

**混合运算模式**：

- 点击 **"混合"** 标签进入混合运算模式
- 至少选择 2 种运算符（加、减、乘、除）
- 遵循 **先乘除后加减** 的标准四则运算优先级
- ⚠️ 混合运算仅支持横式计算

> 💡 **提示**：选择基础运算（加/减/乘/除）时会自动取消混合运算，反之亦然。

### 第三步：设置计算类型（条件显示）

当选中乘法或除法时，会自动显示 **"计算类型"** 卡片：

**乘法相关：**

| 计算类型 | 说明                                                        |
| -------- | ----------------------------------------------------------- |
| 表内乘法 | 九九乘法表范围（1×1 到 9×9），选择后计算范围自动锁定为 1-81 |

**除法相关：**

| 计算类型 | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 表内除法 | 乘法表内的整除运算，计算范围自动锁定为 1-81                  |
| 表内余数 | 乘法表范围内有余数的除法，计算范围自动锁定为 1-81            |
| 一般余数 | 不受乘法表限制，按自定义范围生成有余数的除法（计算范围不锁定） |

> 💡 **提示**：表内乘法、表内除法、表内余数选中后，计算范围会自动锁定为 1-81，不可手动修改。**一般余数**不锁定计算范围。

### 第四步：设置计算范围

在 **"计算范围"** 卡片中，设置数字的最小值和最大值：

- 默认范围：10 ~ 100
- 最大支持范围：1 ~ 999
- 范围锁定时（选中表内类型）自动设为 1 ~ 81

### 第五步：选择出题形式

在 **"出题形式"** 卡片中，选择题目排版方式：

| 形式         | 每页题数 | 布局       | 说明                           |
| ------------ | -------- | ---------- | ------------------------------ |
| **横式计算** | 20 题    | 2列 × 10行 | 适合所有运算类型               |
| **竖式计算** | 12 题    | 3列 × 4行  | 适合加减乘除，混合运算时不可选 |

### 第六步：设置生成页数

在 **"生成页数"** 卡片中，输入需要生成的页数（默认 1 页）。

### 第七步：点击生成

点击底部蓝色的 **"生成"** 按钮，即可进入预览页面查看试卷。

## ⚙️ 功能细节说明

### 题目去重机制

- 使用 Set 数据结构记录已生成题目的"指纹"
- 同一次生成过程中绝不会出现完全相同的题目
- 每次点击"生成"都会产生全新的题目组合

### 难度最大化算法

在"困难"模式下，系统会：

1. 综合考虑**结果大小**、**进位/退位次数**、**位数差距**三个维度
2. 每道题生成 3 个候选，自动选择难度最高的一个
3. 确保每道题都具有足够的挑战性

### 混合运算规则

- 遵循**先乘除后加减**的标准四则运算优先级
- 2 种运算符 → 2 个运算符 + 3 个数字
- 3 种运算符 → 3 个运算符 + 4 个数字
- 4 种运算符 → 4 个运算符 + 5 个数字
- 所有结果保证为**正整数**（大于 0）
- 除法保证精确整除
- 减法中间结果不会出现负数

---

## ⚠️ 注意事项

1. 混合运算模式下**仅支持横式计算**，竖式选项会自动禁用
2. 选择表内乘法、表内除法、表内余数时，**计算范围自动锁定**为 1-81
3. 一般余数类型**不会锁定**计算范围，可自由设置
4. 混合运算**至少需要选择 2 种**运算类型
5. 减法运算的结果保证为**正整数**（不会出现 0 或负数）
6. 除法运算的除数最小为 2（不会出现除以 1 的情况）
7. 普通乘法模式不包含乘数为 1 的题目

## ❓ 常见问题

### Q：为什么生成的题目有重复？

A：系统会自动去重。如果您发现"相似"的题目，那是正常的——系统保证完全相同的题目不重复，但相似题目可能会出现。

### Q：混合运算的优先级是什么？

A：遵循标准数学规则——**先乘除，后加减**。例如 `3 + 4 × 2` 的结果是 11，不是 14。

### Q：如何打印试卷？

A：**H5 网页版**：点击预览页的打印按钮，使用浏览器打印功能。**APP 端**：先保存为 PDF，再用 PDF 阅读器打印。

### Q：为什么除法没有余数？

A：在混合运算模式下，除法仅支持精确整除。如需余数除法，请使用单一除法模式，并选择"表内余数"或"一般余数"类型。

### Q：竖式计算为什么不能选？

A：混合运算模式下仅支持横式计算。请取消混合运算模式，或切换到横式计算。

### Q：计算范围为什么不能修改？

A：当选中了"表内乘法"、"表内除法"或"表内余数"时，计算范围自动锁定为 1-81（九九乘法表范围）。取消这些选项后即可自由设置。
