# Alvin 数页 - 小学数学练习题生成器

计算范围你定义，数学练习随你印

## 在线访问

[https://bi6pah.github.io/my-site/](https://bi6pah.github.io/my-site/)

## 技术栈

- uni-app (Vue 2) + H5
- jsPDF (PDF生成)
- GitHub Pages (托管部署)

## 本地开发

```bash
# 安装依赖
npm install

# 启动 H5 开发服务器
npm run dev:h5

# 构建 H5 生产版本
npm run build:h5
```

## 部署

推送到 `main` 分支会自动触发 GitHub Actions 构建并部署到 GitHub Pages。
