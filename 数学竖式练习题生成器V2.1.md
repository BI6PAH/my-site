# 📐 Alivn数学练习题生成器

小学生初级数学练习题自动生成工具 · 使用说明书

## 一、功能概述

本生成器是一款面向**小学生初级阶段**的数学练习题自动生成工具，支持整数运算（加法、减法、乘法、除法及混合运算），**不含小数和负数**。生成的试卷为标准 A4 格式，每页包含姓名栏和日期栏，可直接打印或保存为 PDF。

**💡 快速定位：**如只需了解基本操作，请直接从"三、设置选项详解"开始阅读。

## 二、界面布局

页面整体分为两个区域：

- **上方控制面板**：包含所有出题设置选项
- **下方预览区**：实时显示生成的试卷页面，以及数据统计面板

## 三、设置选项详解

### 3.1 难度等级

| 选项                         | 说明                                                     |
| :--------------------------- | :------------------------------------------------------- |
| **基础（无进位/借位）**      | 生成的题目保证不存在进位或借位，适合初学者               |
| **标准（60%~70%进位/借位）** | 系统随机产生进位/借位题，比例在 60%~70% 之间（默认选项） |
| **挑战（100%进位/借位）**    | 每道题都包含进位或借位，适合进阶练习                     |

**⚠️ 注意：**难度等级对加法、减法、乘法有效。除法题目难度由除法类型选项控制。

### 3.2 运算类型

**主选项（单选或多选）**

| 选项         | 说明                                 |
| :----------- | :----------------------------------- |
| **加法**     | 两位数以内的整数加法                 |
| **减法**     | 两位数以内的整数减法（结果为正数）   |
| **乘法**     | 整数乘法                             |
| **除法**     | 整数除法（精确整除）                 |
| **混合运算** | 将多种运算组合成一道题目（详见下文） |

**互斥规则：**

- 选中"混合运算"后，加法、减法、乘法、除法四个选项**自动禁用**
- 选中任意一个或多个普通运算时，混合运算**自动禁用**

**子选项（配合主选项显示）**

**乘法子选项：**

- **乘法表范围内练习**：选中后，乘法因子自动限定在 1~9 范围内（九九乘法表）

**除法子选项（支持多选，可同时选中多个）：**

| 选项                       | 说明                                                       |
| :------------------------- | :--------------------------------------------------------- |
| **乘法表范围内的除法**     | 除数和商均在 2~9 范围内，结果为精确整除（如 `36 ÷ 6`）     |
| **乘法表范围内的余数除法** | 被除数 = 除数 × 商 + 余数，余数大于 0（如 `38 ÷ 6 = 6…2`） |

*普通除法：未选中上述任一选项时使用，由计算范围自动控制除法难度。*

### 3.3 混合运算

选中"混合运算"后，会出现额外的"运算组合选择"面板：

- 至少需要选择 **2 种**运算才能生成题目
- 默认选中"加法"和"减法"
- 最终生成 N 种运算的组合题，题目形式为 `数字 运算① 数字 运算② … =`

**混合运算规则：**

- N 种运算 → N+1 个数字，依次排列
- 运算顺序随机打乱
- 支持传入"乘法表范围内练习"和"乘法表范围内的除法"子选项约束
- 混合运算**仅支持横式计算题**（竖式计算题功能在混合运算模式下自动隐藏）
- 题目结果与计算范围限制由算法综合评估，优先选择难度较高的候选题目

### 3.4 计算范围

设置题目中数字的**最小值**和**最大值**。

- 默认范围：最小值 **10**，最大值 **50**
- 输入值须满足：最小值 ≥ 1，最小值 < 最大值

**📌 自动锁定规则：**当选择特定选项时，计算范围自动锁定为 **1~9**，输入框变为只读状态，并显示锁定提示。详见下表：

| 触发条件                                |
| :-------------------------------------- |
| 乘法 + 乘法表范围内练习                 |
| 除法 + 乘法表范围内的除法               |
| 除法 + 乘法表范围内的余数除法           |
| 混合运算中包含乘法 + 乘法表范围内练习   |
| 混合运算中包含除法 + 上述任一除法子选项 |

### 3.5 出题形式

| 形式           | 每页题量 | 说明                                   |
| :------------- | :------- | :------------------------------------- |
| **横式计算题** | 20 题    | 题目以横向排列显示，格式如 `45 + 67 =` |
| **竖式计算题** | 12 题    | 题目以竖式格式显示，包含标准竖式排版   |

**⚠️ 限制：**混合运算模式下，竖式计算题功能**不可用**，系统自动切换为横式并显示提示。

### 3.6 生成设置

**填入页数**：指定生成试卷的总页数，取值范围 **1~50** 页。

## 四、操作流程

### 4.1 生成试卷标准步骤

1. 选择**难度等级**
2. 选择**运算类型**（可单选或多选普通运算，或选择混合运算）
3. 如需细分难度，点击展开**子选项**进行设置
4. 设置**计算范围**
5. 选择**出题形式**
6. 填写**生成页数**
7. 点击 **🔄 生成新题目** 按钮

生成的试卷将实时显示在预览区。

### 4.2 打印 / 保存 PDF

点击 **🖨️ 打印/保存PDF** 按钮，会调出浏览器打印界面：

- 选择**另存为 PDF** 即可将试卷保存为 PDF 文件
- 选择**打印机** 即可直接打印
- 打印时控制面板和统计面板会自动隐藏，只保留试卷内容

**💡 建议：**在打印设置中选择"背景图形"以确保页眉等元素正常显示。

### 4.3 智能去重

系统内置**题目去重机制**，使用 Set 数据结构记录所有已生成的题目组合（包含运算类型、数字和顺序），避免同一张试卷或多次生成时出现重复题目。

## 五、数据统计面板

生成题目后，控制面板下方会显示数据统计区域，包含以下指标：

| 统计项                  | 说明                                                  |
| :---------------------- | :---------------------------------------------------- |
| **实际生成**            | 成功生成的题目数 / 预期总题数                         |
| **本次进位/借位题占比** | 加减乘运算中涉及进位/借位的题目比例（仅普通运算模式） |
| **各运算类型题数**      | 加法、减法、乘法、除法、混合运算各自的题目数量        |
| **去重题目数**          | 全局已生成的不重复题目总数                            |

## 六、竖式格式说明

### 6.1 加减乘竖式

标准竖式排版，数字按位对齐，横线下方留有答题空间。

### 6.2 除法竖式

采用经典**"厂"字形边框设计**：

- 除数（divisor）显示在"厂"字左侧
- 被除数（dividend）位于"厂"字边框内
- 边框由 CSS border 属性绘制，保持与加减乘竖式风格统一

## 七、常见问题

Q1：为什么选中了除法但没有生成除法题目？

请检查计算范围设置。如果范围过小（如 10~20），可能无法生成足够多的整除组合。建议将最大值调大，或切换到"乘法表范围内的除法"子选项。

Q2：混合运算生成失败怎么办？

确保在"运算组合选择"中至少选择了 2 种运算类型。如果持续失败，可能是计算范围过小导致的，请适当扩大范围后重试。

Q3：生成的题目有重复怎么办？

本系统已内置去重机制，相同题目不会在同一批次中重复出现。如需完全不同的新题目，请多次点击"生成新题目"。

Q4：如何生成乘法表（九九表）练习？

选中"乘法" → 勾选"乘法表范围内练习" → 计算范围自动锁定为 1~9。

Q5：打印时页面显示不完整怎么办？

请在浏览器打印设置中检查页边距，建议设置为"无"或最小值，以充分利用 A4 纸张空间。

Q6：如何在混合运算中使用除法？

选中"混合运算" → 在"运算组合选择"中勾选"除法"→ 如需限定除法范围，可同时勾选"乘法表范围内的除法"或"乘法表范围内的余数除法"子选项。
